Section 01

エグゼクティブサマリー

事業は「悪い」のではない。オペレーティングモデルが個人に依存しすぎている。 本物のアセットを仕組みに変換することが最優先課題である。

診断結論

内定率82%、累計900名の支援実績、ACs 300人のコミュニティ。 これらは他社が簡単に模倣できない本物の競争優位性である。 問題は事業の質ではなく、一人の人間にすべてのオペレーションが集中している構造にある。 仕組み化（Systemization）を実行すれば、事業価値は現在の2-3倍に跳ね上がる。

1

資産は本物

内定率82%は業界最高水準。900名実績とACs 300人ネットワークは、コンサル就活市場における稀有なアセット。

2

構造的ボトルネック

CEO = CTO = CMO = 営業マネージャー。一人5役の体制は短期的には効率的だが、スケール不可能。

3

仕組み化が最優先

属人的なオペレーションをシステムとプロセスに転換する。AIとCRMの活用が鍵。

4

2026年末 目標年商

ポートフォリオ再構築により、年商1.5-2億円の到達が現実的に見えるポジションにある。

Section 03

SWOT分析

内部環境と外部環境の交差点から、戦略オプションを導出する。

Strengths（強み）

  • 内定率82%という圧倒的な実績
  • 累計900名の支援実績と信頼
  • ACs 300人のOB/OGネットワーク
  • コンサル業界への深い理解と人脈
  • AI/テクノロジーへの高い感度

Weaknesses（弱み）

  • 創業者（田中）への属人依存
  • マーケティング基盤の未整備
  • テクノロジー基盤の未成熟
  • 組織としてのスケーラビリティ欠如
  • セールスプロセスの非標準化

Opportunities（機会）

  • AI面接SaaSによる市場創出
  • コンサルタイムズのメディア化
  • ACs活用による講師・メンター拡充
  • MBA受験・海外大学生市場への拡張
  • CRM/AI自動化による業務効率化

Threats（脅威）

  • 競合の増加（選択肢の爆発的増加）
  • 価格競争の激化
  • 情報の非対称性の縮小
  • コンサル採用市場のマクロ変動
  • Key Person離脱リスク

クロスSWOT戦略マトリクス

SO戦略（積極攻勢）

強み x 機会

  • 内定率82%・900名実績データを活用したコンテンツマーケティング
  • ACs 300人から優秀な講師をリクルートし、指導体制を分散化
  • コンサルタイムズを業界No.1メディアに育成
  • 実績データを基盤にAI面接SaaSを開発
WO戦略（弱点補強）

弱み x 機会

  • AI面接SaaSで属人的な面接指導を自動化
  • CRM/マーケティング自動化で営業プロセスを標準化
  • テクノロジー投資により、少人数でもスケール可能な体制構築
  • 外部マーケターの起用でマーケ機能を補完
ST戦略（差別化防衛）

強み x 脅威

  • 「内定率82%」を前面に出し、実績ベースで競合と差別化
  • コンサルタイムズで業界オピニオンリーダーの地位を確立
  • 価格競争に巻き込まれない高付加価値ブランド構築
  • ACs コミュニティの厚さで参入障壁を形成
WT戦略（撤退/ピボット）

弱み x 脅威

  • 最悪シナリオ：教育事業売却 + AI事業へのピボット
  • AI面接SaaS に経営資源を集中し、EdTech企業として再定義
  • 教育事業の売却資金でAI事業の開発を加速
  • 属人性を逆手に「創業者ストーリー」として売却価値を最大化
Section 04

イシューツリー

「なぜスケールしないのか？」を構造的に分解する。

事業がスケールしない
問題 全オペレーションが山本一人に集中している
原因 営業プロセスが標準化されていない（暗黙知の蓄積）
施策 CRM + 営業プレイブックの整備、ACs への委任
原因 マーケティングが属人的なSNS運用に依存
施策 コンテンツマーケ自動化、マーケ外注パートナー契約
問題 収益源が単一事業に偏っている
原因 新規事業の開発リソースが確保できない
施策 AI面接SaaS MVP開発（外部委託 + 社内AI活用）
問題 事業売却時の評価額が低い
原因 SaaS的なリカーリング収益がない（教育事業倍率適用）
施策 SaaS MVPを付帯し、SaaS倍率での評価を実現

Section 07

組織のボトルネック

山本 = CEO + CTO + CMO + 営業マネージャー + 新規事業責任者。 この構造を解体しない限り、成長はない。

山本
一人5役
CEO（経営判断）
CTO（技術開発）
CMO（マーケ）
営業マネージャー
新規事業責任者

営業の仕組み化

CRM + 営業プレイブック整備。リード獲得からクロージングまでのプロセスを標準化し、ACs メンバーに段階的に委任。山本が全件対応する必要をなくす。

マーケの外注化

SNS運用・コンテンツ制作・広告運用を外部パートナーに委託。山本は戦略方針の策定のみに集中。コンサルタイムズの編集も外注可能。

AI面接SaaS 外部委託

開発は外部エンジニアに委託し、山本はプロダクトオーナーとしてドメイン知識の提供に専念。Claude Code等のAIツールも活用して開発速度を最大化。

ACs への委任

300人のACs ネットワークから、面接指導・メンタリングの担い手を育成。段階的に権限委譲し、山本なしでも回る指導体制を構築。

Section 09

撤退基準（Kill Criteria）

各事業の「やめる条件」を事前に定義する。 感情ではなくデータで撤退判断を行うための基準。

Strategists（教育事業）

  • 年間売上が500万円を3ヶ月連続で下回る
  • 内定率が60%を下回る（ブランド価値毀損）
  • 新規リード数が月10件未満が6ヶ月継続
  • 田中の指導時間が月100時間超で改善の兆しなし

AI面接SaaS

  • MVP後3ヶ月で有料ユーザー10社未満
  • 月間チャーンレートが15%超を3ヶ月連続
  • 開発費が累計500万円を超えてPMF未達成
  • 類似製品が大手から出て差別化困難

Claude Code コミュニティ

  • 3ヶ月でアクティブ会員30人未満
  • 月額売上が20万円を下回り改善策なし
  • 山本の運営工数が月20時間超で自動化困難
  • コミュニティ満足度NPS 30以下

人材紹介

  • 年間紹介成約が5件未満
  • 1件あたりの営業工数が40時間超
  • 候補者プールが100名を下回る
  • 紹介先企業からのクレーム率10%超
